# 한국어 띄어쓰기는 어려워

> "한국어를 사용하는 사람들이 가장 궁금해하는 것 1위, 그리고 국어 전문가조차 업무 상황에서 가장 고심하는 것 역시 '띄어쓰기'입니다."

어떤 표현을 띄어쓰기 없이 입력하면, **우리말샘 사전**과 **한글 맞춤법 조항**을 결합하여 판단 근거를 한눈에 보여 주는 도구입니다.

## 바로 사용하기

**[korean-spacing-v0.1.0.zip 다운로드](https://github.com/sophie-linguist/korean-spacing/releases/download/v0.1.0/korean-spacing-v0.1.0.zip)**

1. 위 링크에서 zip을 다운로드합니다.
2. 압축을 풀고 **korean-spacing.exe**를 더블클릭합니다.
3. 검색창에 궁금한 표현을 입력하면 끝입니다.

> Windows 보안 경고가 뜨면 "추가 정보" → "실행"을 누르세요.

---

## 왜 만들었나

**국어생활 상담 유형 압도적 1위: 띄어쓰기 (35.48%)**

국립국어원의 연간 수십만 건에 달하는 상담 자료를 분석한 결과, 띄어쓰기 관련 문의는 전체의 35.48%(201,282건)로 압도적 1위입니다. 전문가조차 정확한 자문을 위해 매번 사전을 찾고, 규정 해설을 검토하고, 품사 결합 규칙을 대조해야 합니다.

**"이 복잡한 판단 근거들을, 체계가 먼저 분석하여 친절하게 보여 줄 수는 없을까?"**

이 실무적 요구에서 도구가 출발했습니다.

## 핵심 기능

### 1. 동음이의어 대비 안내

> 입력: `법대로`, `아는데`, `회의중`

'대로'가 체언 뒤 붙여 쓰는 조사(제41항)인지, 관형사형 뒤 띄어 쓰는 의존명사(제42항)인지 — 두 해석을 동시에 비교하여 보여줍니다.

### 2. 결합형(보조 용언) 분리 및 규칙 검증

> 입력: `할만하다`

'할(본용언) + 만하다(보조용언)' 구조를 자동 분리하여, 제47항에 따른 원칙(할 만하다)과 허용(할만하다)의 근거를 함께 안내합니다.

### 3. 미등록 전문 용어·고유 명사 인식

> 입력: `인공지능위원회`

사전에 없는 합성어라도 기관 접미사 신호를 감지하여, "단어별로 띄어 씀이 원칙이나 붙여 쓸 수 있음"(제49·50항)을 안내합니다.

## 기술 구성

| 구성 요소 | 설명 |
|-----------|------|
| **자료 기반** | 우리말샘 사전(JSON) → SQLite(`dict.db`) 색인화 |
| **핵심 논리** | 자모 기반 머리 검증 + 꼬리 사전 활용 (형태소 분석기 없이 가벼운 단어 경계 탐색) |
| **규정 체계** | 한글 맞춤법 제2항 + 제5장(제41~50항) 해설 내장 |
| **화면** | pywebview 웹 UI (기본) / Tkinter (레거시) |
| **배포** | PyInstaller onefile exe + dict.db 사이드카 |

### 설계 원칙

이 도구는 **자동 교정기가 아닙니다.** 판단 근거(사전·품사·조항)를 펼쳐 보여 주고, 최종 결정은 사람에게 맡깁니다. 동형이의어처럼 도구가 판별할 수 없는 경우에도 사용자를 오도하지 않고, 가능한 후보를 정리해 보여줄 뿐입니다.

---

## 현재의 한계

"판단이 모호하면 침묵한다"는 고신뢰 원칙하에 안전 범위를 지정하였습니다.

- **불규칙 활용**: 규칙 활용만 지원. ㄹ/ㅂ/ㄷ/르/ㅎ 불규칙(예: 알다 → 아는 체)은 차기 과제.
- **문장 단위 분석**: 단어 및 짧은 표현 단위에 특화. 문장 분석은 형태소 분석기 연동으로 확장 예정.

## 개발자용

### 설치

```bash
pip install -r requirements.txt
```

### 환경변수

`.env.example`을 복사해 `.env`를 만든 뒤 값을 설정합니다.

| 변수 | 필수 | 설명 |
|------|------|------|
| `KOREAN_SPACING_DB_PATH` | 선택 | `dict.db` 경로. 미설정 시 현재 폴더 또는 exe 옆 `dict.db` 사용 |

### 실행

```bash
# 웹 UI (기본)
python -m shell.webui.app

# Tkinter UI (레거시)
python -m shell.gui.app
```

### 사전 인덱스 빌드

```bash
python build/build_index.py \
  --source "전체 내려받기_우리말샘_json_20260603" \
  --output "dict.db" \
  --schema "build/schema.sql"
```

### exe 빌드

```powershell
powershell -ExecutionPolicy Bypass -File .\build\build_exe.ps1 -DictDbPath "dict.db"
```
